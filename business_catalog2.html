📝 Content Management Systems (CMS)

WordPress A free, open-source content management system (CMS) used to create and manage websites and blogs without needing to code. It powers over 40% of all websites worldwide.

Pimcore An open-source digital experience platform (DXP) combining CMS, PIM (Product Information Management), DAM, and e-commerce capabilities in a single platform.

💻 Programming Languages, Platforms & Frameworks
LogoProduct / ServiceDescription
Angular / AngularJS A TypeScript-based open-source front-end web application framework developed by Google. Angular is the modern version (v2+); AngularJS (v1.x) was the original JavaScript framework used to build dynamic single-page applications.
React / React Native / React Router React is a JavaScript library by Meta for building user interfaces. React Native extends it for mobile apps (iOS/Android). React Router is a routing library for React web applications.
Vue.js A progressive open-source JavaScript framework for building user interfaces and single-page applications. It is known for its gentle learning curve, flexibility, and reactive data binding.
Nuxt.js A Vue.js-based framework for building server-rendered, statically generated, or single-page web applications. It simplifies universal (SSR) Vue app development with powerful conventions.
Next.js A React-based framework by Vercel for building full-stack web applications. It supports server-side rendering (SSR), static site generation (SSG), and API routes out of the box.
Node.js An open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside a browser. It is widely used to build scalable, event-driven server-side applications and APIs.
Flutter An open-source UI toolkit by Google for building natively compiled, multi-platform applications (iOS, Android, Web, Desktop) from a single Dart codebase.
Python A popular, versatile, high-level programming language known for its clear syntax and readability. Widely used in web development, data science, machine learning, automation, and scripting.
PHP A widely-used open-source server-side scripting language especially suited for web development. It powers many major CMS platforms and is the foundation of a large portion of the web.
Ruby / Ruby on Rails Ruby is a dynamic, open-source programming language focused on simplicity. Ruby on Rails (Rails) is a web application framework built on Ruby that follows the MVC pattern, enabling rapid web development.
Java A widely-used, object-oriented programming language and platform known for its "write once, run anywhere" portability. It is used extensively in enterprise software, Android development, and web backends.
C++
 C++ / C++ Qt C++ is a general-purpose programming language extending C with object-oriented features. Qt is a cross-platform C++ framework for developing desktop, mobile, and embedded applications with rich GUIs.
C#
 C# (.NET) A modern, object-oriented programming language by Microsoft used for building Windows applications, web services, and cross-platform apps with the .NET ecosystem. Clients available for both Tracking and Analytics APIs.
Dart A client-optimized, open-source programming language by Google, primarily used for building mobile (Flutter), web, and server applications. It compiles to ARM/x64 native code or JavaScript.
Objective-C
 Objective-C A general-purpose, object-oriented programming language that was the primary language used for Apple macOS and iOS development before Swift. Still used in legacy Apple platform codebases.
Laravel A free, open-source PHP web application framework with expressive, elegant syntax. It provides tools for routing, authentication, caching, queues, and more, making web development enjoyable.
Symfony A PHP web application framework and a set of reusable PHP components. Used by many large-scale applications and as a foundation for other frameworks like Laravel and Drupal.
Django A high-level Python web framework that encourages rapid development and clean, pragmatic design. It follows the DRY (Don't Repeat Yourself) principle and includes batteries for authentication, ORM, and admin.
Hugo An open-source static site generator written in Go. It is one of the fastest static site builders available, ideal for blogs, documentation sites, and portfolios with minimal configuration.
RudderStack An open-source customer data platform (CDP) for collecting, routing, and transforming event data from multiple sources to various data warehouses and analytics tools.
N8N A fair-code, open-source workflow automation tool that allows users to connect apps, services, and APIs without deep coding knowledge. Think of it as a self-hostable alternative to Zapier.
GTM (Server-side) Google Tag Manager (Server-side) is a tag management system where tags are processed on a server rather than the client browser, enabling better privacy, performance, and data control.
Redux Beacon
 Redux Beacon A JavaScript library for integrating analytics into Redux and ngrx applications. It allows developers to track application events by mapping Redux actions to analytics events.
Xamarin
 Xamarin A Microsoft open-source platform for building iOS, Android, and Windows apps using C# and .NET. It enables code sharing across mobile platforms while maintaining native performance and look-and-feel.
iOS & Android
 iOS & Android (Native SDKs) Native mobile operating system platforms by Apple (iOS) and Google (Android). Matomo provides official SDKs for tracking user behavior within native mobile apps on both platforms.
MCP
 MCP (Model Context Protocol) An open protocol by Anthropic that standardizes how AI models (like Claude) connect to external tools, data sources, and services. The Matomo MCP integration enables AI assistants to query analytics data directly.
